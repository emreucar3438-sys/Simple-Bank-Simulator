# 🏦 Simple Python Bank Simulator

A terminal-based banking application developed using **Object-Oriented Programming (OOP)** principles in Python. This project features persistent data storage, allowing user balances to be saved and reloaded automatically.

## ✨ Key Features
- **OOP Architecture:** Managed via the `Bank_account` class for clean and maintainable code.
- **Data Persistence:** Automatically saves the account balance to a `hesap_ozeti.txt` file after every transaction.
- **Secure File Handling:** Uses `try-except` blocks to handle missing or corrupted data files gracefully.
- **Interactive Menu:** A dynamic command-line interface for checking balance, depositing, and withdrawing funds.

## 🛠️ Technologies Used
- **Language:** Python 3.x
- **Storage:** Local File I/O (Text-based storage)
